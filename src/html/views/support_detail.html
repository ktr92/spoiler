Гарантийные обязательства

Компания SK Server Technologies предоставляет срок гарантийного обслуживания серверов, СХД и рабочих станций торговой марки SK Gelios на 3 года с даты продажи. Гарантия распространяется на все компоненты сервера, СХД и рабочей станции, включая вентиляторы охлаждения, накопители и т.д. При возникновении неисправности в течение гарантийного срока необходимо обратиться в сервисный центр компании.

  • Компания SK Server Technologies обязуется производить гарантийный ремонт поставляемой техники при условии соблюдения заказчиком правил эксплуатации.
  • В течение 1-го года гарантийного обслуживания - выезд инженера сервисной службы для выполнения гарантийных работ по диагностике и ремонту вышедшего из строя оборудования осуществляется либо в день обращения, либо на следующий рабочий день.* Если выполнения гарантийных работ по диагностике и ремонту оборудования будет невозможно на месте эксплуатации, компания SK Server Technologies транспортирует оборудование в сервисный центр для полной диагностики и ремонта.
  • В течение 2-го и 3-го года - гарантийное обслуживание выполняется в сервисном центре компании SK Server Technologies.
  • Продолжительность гарантийного ремонта не должна превышать 40 рабочих дней. Гарантийный срок продлевается на время нахождения Оборудования в ремонте.
  • В случае проявления у приобретенного оборудования существенного недостатка (т.е. недостатка, который не может быть устранен) производится замена неисправного изделия на аналогичное исправное изделие (при обязательном наличии заключения сервисного центра о существенности недостатка изделия).
